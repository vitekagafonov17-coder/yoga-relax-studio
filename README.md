# Студия йоги "Relax"

Многостраничный сайт студии йоги с расписанием занятий, профилями инструкторов и формой записи.

## Функционал
- Главная страница с описанием студии
- Расписание групповых и индивидуальных занятий
- Профили инструкторов с подробной информацией
- Форма записи на пробный урок
- Адаптивный дизайн

## Технологии
- HTML5
- CSS3
- JavaScript
- Font Awesome иконки
- Google Fonts

## Как запустить локально
Откройте файл `index.html` в браузере или используйте расширение Live Server в VS Code.
